# SecEmailCMD

Данная программа предоставляет вам доступ к временной почте из любого терминала!

# Установка Termux:
```
apt update
apt upgrade
apt install python
apt install git
git clone https://github.com/AdolfGitHuber/SecEmailCMD
pip install requests
```
# Запуск:
```
cd SecEmailCMD
python main.py
```
